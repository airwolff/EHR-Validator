Severity, defined once

Plausibility of the datum — not patient survivability. SpO2 > 100 is critical because it is impossible data.

The thesis

For fixed, well-specified rules, deterministic Python is the correct tool — faster, cheaper, auditable. The LLM exists only for what rules can't reach: unstructured text, cross-field contradictions, patterns across records.

Limit 1 — pinned by tests

Not an injection defence. An instruction riding inside the note gets cited verbatim, so the guard keeps it.

Limit 2 — pinned by tests

Verbatim ≠ faithful. A model can cite "dysuria" out of "denies dysuria" — so a human adjudicates the worklist.

I did not let a language model decide control flow over patient data.

Way Forward

Four items

Adjudicator agent deferred, not dropped

When the clinical and identity specialists conflict on the same record, a third call judges the disagreement — a job rules genuinely can't do.

Real-corpus caveats

Everything here is synthetic and planted. Real notes are messier — the grader and the bias framing both need rework before this touches anything real.
